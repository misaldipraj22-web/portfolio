## 📌 About

Hi, I'm **Dipraj Misal** — a passionate **Frontend Engineer** from Jalna, Maharashtra with **2 years of experience** building beautiful, performant web experiences.

This portfolio showcases my skills, projects, and journey as a frontend developer. Built from scratch with a focus on modern design aesthetics, smooth animations, and clean code.

---

## ✨ Features

- 🎨 **Dark Theme** — Premium glassmorphism dark UI design
- ✍️ **Typing Animation** — Dynamic text animation on the hero section
- 🌌 **Particle Canvas** — Interactive animated background particles
- 📱 **Fully Responsive** — Works on all screen sizes
- ⚡ **Pure Vanilla** — No frameworks, just HTML + CSS + JS
- 🎯 **Smooth Scroll** — Seamless navigation between sections
- 💌 **Contact Form** — With built-in validation
- 🔍 **SEO Optimized** — Meta tags, semantic HTML5 structure
